### Acrylic shader
アクリルのような質感のシェーダです。
視線ベクトルと法線ベクトルが平行な程透明度を高く、垂直な程透明度を低くしています。

### stainedglass shader
ステンドグラス風のシェーダです。
画像をグレースケール化し、黒に近い値（0.2未満）は不透明に、それ以外は透明度を上げました。
